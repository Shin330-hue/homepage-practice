# 仕様書

## 概要

動画クリエイターが自身の作品・スキルを紹介するためのポートフォリオサイト。

## ページ構成

### 1. ヒーローセクション
- フルスクリーン表示
- 背景画像（暗めのオーバーレイ付き）
- キャッチコピー
- 「View My Work」CTAボタン

### 2. ナビゲーション
- 固定ヘッダー（スクロールで背景変化）
- ロゴ（左）
- リンク（右）: About / Works / Contact

### 3. About セクション
- プロフィール画像（左）
- 自己紹介文（右）
- 2カラムレイアウト（モバイルで縦積み）

### 4. Works セクション
- 作品カード（グリッドレイアウト）
- 各カード:
  - サムネイル画像（16:9）
  - ホバーで再生アイコン表示
  - カテゴリ / タイトル / 説明

### 5. Skills セクション
- スキルタグ（横並び・中央揃え）
- アイコン + テキスト

### 6. Contact セクション
- SNSリンク（Twitter, YouTube, Email）
- アイコンボタン

### 7. フッター
- コピーライト

## デザイン仕様

### カラーパレット
| 変数名 | 値 | 用途 |
|--------|-----|------|
| --bg-dark | #0a0a0e | 背景 |
| --bg-card | rgba(25,25,35,0.6) | カード背景 |
| --primary | #ff3366 | アクセント（CTA、ホバー） |
| --secondary | #00f2ff | サブアクセント（見出し） |
| --text-main | #ffffff | メインテキスト |
| --text-muted | #8892b0 | サブテキスト |

### フォント
- Outfit（Google Fonts）

### アニメーション
- Framer Motion使用
- フェードイン + 上方向スライド
- スクロール連動表示
- ホバーエフェクト

## レスポンシブ対応
- ブレークポイント: 768px
- モバイル時: 1カラム表示
