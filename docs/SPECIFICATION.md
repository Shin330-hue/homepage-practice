# 仕様書

## 概要

岐阜を拠点とする映像制作クリエイター「K's Movie Producet」のポートフォリオサイト。
ビジネスからバラエティまで幅広いジャンルの動画編集・サムネイル制作実績を紹介。

## サイト構成

### ページ一覧（全7ページ）

1. **トップページ** (`/`)
   - Hero、挨拶、動画セクション、サムネイルセクション、問い合わせフォーム
2. **動画作品ページ** (4種類)
   - ビジネス縦型 (`/video/business/vertical`)
   - ビジネス横型 (`/video/business/horizontal`) ※YouTube埋め込み実装済み
   - バラエティ縦型 (`/video/variety/vertical`)
   - バラエティ横型 (`/video/variety/horizontal`)
3. **サムネイル作品ページ** (2種類)
   - ビジネス (`/thumbnail/business`)
   - バラエティ (`/thumbnail/variety`)

## セクション詳細

### 1. ナビゲーション
- 固定ヘッダー（スクロールで背景・影変化）
- ロゴ「K's Movie Producet」（左）
- リンク（右）: 動画 / サムネイル / お問い合わせ
- モバイル時: リンク非表示（シンプル化）

### 2. Heroセクション
- **レイアウト**: 左テキスト / 右画像（2カラム）
- **見出し**: 「映像は"魂"を映し出す」
- **説明文**: 岐阜ベースの映像制作アピール
- **CTA**: 「お問い合わせ →」ボタン
- モバイル時: 縦積み、中央揃え

### 3. 挨拶セクション
- **レイアウト**: 左テキスト / 右画像（2カラム）
- **見出し**: 「ご挨拶」
- **メッセージ**: 「ワクワクする体験」をテーマにした情熱的なメッセージ
- モバイル時: 縦積み、中央揃え

### 4. 動画セクション
- **レイアウト**: 2カラムグリッド（左=ビジネス / 右=バラエティ）
- 各カラム:
  - 縦型・横型の2つのサムネイル
  - クリックで詳細ページへ遷移
- モバイル時: 1カラム縦積み

### 5. サムネイルセクション
- **レイアウト**: 2カラムグリッド（ビジネス / バラエティ）
- 各カード:
  - サムネイル画像（16:9比率）
  - タイトル
  - クリックで詳細ページへ遷移
- モバイル時: 1カラム縦積み

### 6. 問い合わせフォーム
- **フィールド**:
  - 名前（必須）
  - メールアドレス（必須）
  - 電話番号
  - お問い合わせ内容
- **デザイン**: 白背景カード、シャドウ付き
- **送信機能**: フロントエンド実装済み（バックエンド連携は今後）

### 7. フッター
- コピーライト: © 2025 K's Movie Producet

## 作品ページ詳細

### 動画作品ページ
- **共通要素**:
  - ページタイトル
  - 「← トップに戻る」リンク
  - 作品グリッド表示
- **ビジネス横型ページの特別仕様**:
  - YouTube埋め込みプレーヤー（4本）
  - レスポンシブ対応（16:9比率維持）
  - 動画タイトル表示
- **その他ページ**: プレースホルダー表示（今後コンテンツ追加予定）

### サムネイル作品ページ
- グリッドレイアウト
- 作品カード（サムネイル + タイトル + 説明）
- プレースホルダー表示（今後コンテンツ追加予定）

## デザイン仕様

### カラーパレット（ライトモード）
| 変数名 | 値 | 用途 |
|--------|-----|------|
| --bg-dark | #ffffff | 背景 |
| --bg-card | #f5f5f5 | カード背景 |
| --primary | #333333 | メインテキスト |
| --secondary | #666666 | サブテキスト |
| --accent | #e74c3c | アクセント（CTA、リンクホバー） |
| --text-main | #333333 | テキスト |
| --text-muted | #666666 | サブテキスト |

### フォント
- **メイン**: Outfit（Google Fonts）
- **日本語フォールバック**: Noto Sans JP

### アニメーション
- **ライブラリ**: Framer Motion
- **パターン**:
  - フェードイン + 上方向スライド（`fadeInUp`）
  - スクロール連動表示（`whileInView`）
  - ホバーエフェクト（transform、opacity）
- **特別機能**: ページ遷移時のスクロールリセット

## 技術仕様

### YouTube埋め込み
- **方法**: `<iframe>` タグで埋め込み
- **スタイリング**:
  - 16:9比率維持（padding-bottomハック）
  - レスポンシブ対応
  - border-radius: 8px
- **パラメータ**: autoplay無効、フルスクリーン許可

### ルーティング
- **形式**: `/video/:category/:type`, `/thumbnail/:category`
- **パラメータ**:
  - category: `business` | `variety`
  - type: `vertical` | `horizontal`
- **ScrollToTop**: ルート変更時に自動的に `window.scrollTo(0, 0)`

## レスポンシブ対応

### ブレークポイント
- **900px以下**: Hero/Greetingを縦積み、動画グリッド調整
- **768px以下**: フルモバイル対応（1カラム表示）

### モバイル時の挙動
- ナビゲーションリンク非表示
- 全セクション縦積み
- 動画サムネイルサイズ調整
- 問い合わせフォームパディング調整
